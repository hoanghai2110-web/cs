Privacy Policy

Last updated: {{ current_year }}

Overview

SMS Receiver is a demonstration platform that provides virtual phone numbers and generated SMS messages for testing purposes. This privacy policy explains how we handle information in our sandbox environment.

Information We Collect

Automatically Collected Information

  • • Basic server logs including IP addresses and access times
  • • Browser type and device information
  • • Pages visited and navigation patterns
  • • Referral sources and session duration

Information We Do NOT Collect

  • • No personal identification information
  • • No real phone numbers or contact details
  • • No user accounts or registration data
  • • No payment or financial information
  • • No real SMS message content

How We Use Information

The limited information we collect is used solely for:

  • • Maintaining platform functionality and performance
  • • Monitoring system security and preventing abuse
  • • Understanding usage patterns to improve the service
  • • Generating anonymous analytics for platform optimization

Data Storage and Security

Generated Content

All phone numbers and SMS messages are algorithmically generated on-demand. No persistent storage of generated content occurs, and all data refreshes daily using deterministic algorithms.

Security Measures

  • • HTTPS encryption for all data transmission
  • • Regular security updates and monitoring
  • • No permanent storage of sensitive information
  • • Isolated sandbox environment with no external connections

Third-Party Services

This platform may use:

  • • CDN services for faster content delivery (TailwindCSS)
  • • Basic web analytics for usage statistics
  • • Cloud hosting infrastructure for platform operation

These services operate under their own privacy policies and may collect standard web analytics data.

Your Rights

Since this is a demonstration platform that generates synthetic data:

  • • No personal data is collected that requires deletion rights
  • • All generated content is fictional and not associated with real individuals
  • • You can stop using the platform at any time without data concerns
  • • No account creation means no stored personal preferences or data

Important Disclaimers

Demonstration Purpose Only

This platform is for testing and demonstration purposes only. Do not use it for any real communications or production applications.

No Real Telecommunications

No actual phone numbers or SMS services are involved. All content is generated and fictional.

No Warranty

The platform is provided "as is" without warranties of any kind for demonstration purposes.

Questions About Privacy?

This platform operates as a simple demonstration tool with minimal data collection. For technical questions, please refer to our help documentation.
